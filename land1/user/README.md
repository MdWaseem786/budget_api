# Aranpani

Aranpani is a Flutter-based platform for land development planning and verification.

## Repository Structure

- `user/` – User-facing Flutter application for submitting land development plans, images, and location details.
- `admin/` – Admin Flutter application for reviewing, verifying, and managing submitted plans.

## Tech Stack
- Flutter
- Firebase Authentication
- Firestore
- Cloudinary (image storage)
- Location Services
